Ejemplos del Mundo Real – Programación Orientada a Objetos (POO)
---------------------
Este repositorio contiene ejemplos sencillos en Python que aplican los conceptos fundamentales de la Programación Orientada a Objetos (POO) mediante situaciones del mundo real. Los ejemplos están diseñados para facilitar la comprensión de clases, atributos, métodos y la interacción entre objetos.

tienda.py – Sistema de Tienda (Carrito de Compras)
---------------------
Este programa modela una situación del mundo real donde una persona agrega productos a un carrito de compras. No se manejan pagos ni cálculos avanzados, ya que el objetivo es demostrar claramente el uso de la Programación Orientada a Objetos.

Objetivo

Representar productos mediante una clase
Permitir que una persona agregue productos a su carrito
Mostrar los productos agregados

Clases utilizadas

Producto: representa un producto con un nombre
Persona: representa a una persona que puede agregar productos a un carrito
